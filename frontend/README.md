# FinStream Frontend

React + Vite client for the FinStream dashboard.

## Requirements
- Node.js 18+
- npm

## Install
```bash
cd /home/runner/work/FinStream/FinStream/frontend
npm install
```

## Run (Development)
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```

## Lint
```bash
npm run lint
```

## Environment Variable
Create a `.env` file in `/home/runner/work/FinStream/FinStream/frontend` if needed:
```env
VITE_API_URL=http://localhost:8080
```
